# RentSplitter.org

A free rent splitting calculator for roommates. Handles equal splits, square-footage-based splits, income-proportional splits, and custom weight configurations. No signup, no tracking, no drama.

## Features

- **4 Splitting Modes**: Equal, By Square Footage, By Income, Custom Weights
- **Roommate Management**: Add/remove roommates dynamically
- **Private Bathroom Premium**: Auto-applies 15% weight bonus for ensuite baths
- **Couple Support**: Adjusts split proportionally for shared rooms
- **Utilities Included**: Factor in utilities and extras alongside rent
- **No Data Collection**: All calculations run client-side

## Tech Stack

- Pure static HTML/CSS/JS
- No frameworks, no build step
- Deployed on Vercel
- Mobile-first responsive design
